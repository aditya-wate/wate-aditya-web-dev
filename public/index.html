Cloud services work on pay per use model which results in savings for the client subscribing to the service provider. To deal with this infrastructure the cloud services deploy multiple servers on one physical server as virtual machines (VM). This is where the security issue is rooted; even if the cloud service provider is trusted and a non-malicious entity, the co-located machines in the same physical infrastructure may not have good intentions.
